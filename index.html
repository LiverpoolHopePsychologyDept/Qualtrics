Accessing Qualtrics

  • To create an account with Liverpool Hope Qualtrics you should follow this link and enter your @hope.ac.uk email address along with the access code that you can find on the Departmental Moodle
  • Log in to your account using this link https://liverpoolhope.eu.qualtrics.com/

What is Qualtrics?

Qualtrics is a GDPR compliant cloud based software that allows users to create professional quality research studies. There are many features but some of key elements that you will use are:

  • Different question types, suited to the type of data that you want to collect
  • Scoring methods which will simplfy the output of your data
  • Pre-Built Libraries of templates and full questionnaires
  • Different distribution options
  • Advanced survey tools such as Skip Logic and Embedded Data
  • Scoring options
  • Post survey options
